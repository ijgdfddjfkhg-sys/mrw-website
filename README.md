## 🚀 كيفية التشغيل

1. **تحميل الملفات:**
   ```bash
   git clone [repository-url]
   cd mrw
   ```

2. **إضافة الصور:**
   - ضع صور الشعار في مجلد `images/`
   - أضف صور المنتجات والخدمات
   - تأكد من تطابق أسماء الملفات مع المراجع في الكود

3. **تشغيل الموقع:**
   - افتح `index.html` في المتصفح
   - أو استخدم خادم محلي مثل Live Server
